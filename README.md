# Sticky Lingo

A Spanish vocabulary app for English speakers, built around a simple idea: you already know more Spanish than you think. Swipe through cognates, patterns, and common verbs — emoji-first, offline, no games.

Cards are organized by cognate-recognition difficulty: easy associations → formal English cognates → pattern conversions → verbs.

**Stack:** Expo (managed) + TypeScript, Expo Router, Zustand, expo-sqlite, react-native-reanimated
**Status:** V0 — read-only swipe experience. No audio, no gamification, no accounts.
